## 📌 Overview  
This project is a cloud-based retail analytics dashboard built using Microsoft Azure. It allows users to explore customer purchasing behavior and analyze retail data through an interactive web application.

---

## 🚀 Features  

- Login system (username, password, email)  
- Dashboard with key metrics (sales, spending, units)  
- Search by Household ID  
- Charts for sales trends and regions  
- Basket analysis (common product pairs)  
- Machine learning (high vs low spender prediction)  
- Churn detection (low-spending customers)  

---

## 🗄️ Data  
The project uses retail datasets including:
- Transactions  
- Households  
- Products  

Data is loaded and processed within the Azure-hosted application.
